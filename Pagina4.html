Módulos que puedes usar: Módulos de Gas, Sensores de luz, Sensores de proximidad, teclados, pantallas, displays, motores, sensores de humedad, etc...


La mayoría de sensores y módulos pueden trabajar en conjunto, conectados al mismo Arduino. Este en concreto es el Modulo HC-SR04. Existen muchos mas pero aquí solo te presentamos visualmente uno. Dependerá de lo que quieras usar y que funciones va a cumplir tu proyecto. Existe el sensor infrarrojo, un motor paso a paso, sensor de humedad, sensor de temperatura, modulo de memoria SD, modulo bluetooth HC-05, pantalla LCD con modulo para interfaz I2C, pantallas oled, Modulo L298N (sirve para hacer carros a bluetooth), Modulo de Relé, Sensor ultrasónico, modulo I2C (sirve para los displays), display de 7 segmentos, Modulo DS3231 (este modulo sirve para hacer relojes en tiempo real y que no olviden la hora cuando se quite la energía), Joysticks, etc...
